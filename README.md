# CommanderCampsite

CommanderCompsite is a Kotlin-based Andriod appplicatin that help the user stay orgarnized when going camping so that she or he does not have to carry unecessary stuff and enjoy camping 
The app prompt the user to add a gear(e.g item name , category, quantity, comments
The progect also implements GitHUB version controlActions to ensure a robust and maintainable codebase

## Features

it store camping gear and food supplies
-allows user to categorise the camping gear
-it provide detailed checklist
-Includes a reset button to clear input and results
-Simple, clean and user-friendly interface

## How to use

 1.Open the application
 2.Enter a camping gear under item name, category, quantity, comments
 3.Tap "add gear" button
 4.Tap "Reset" to clear input and try again

 NOTE BE: The application currently accepts only predefined time categories as input as app logic




## Build with

-Kotlin
-Android Studio
-Jackpack Compose
